### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/fregues-mp/teeworlds-analyze.git
   ```
2. Navigate to the project directory:
   ```bash
   cd teeworlds-analyze
   ```
3. Install the dependencies:
   ```bash
   npm i
   ```
4. Read servers/README.md
   
5. Run
   ```
   node multbot.js
   ```
